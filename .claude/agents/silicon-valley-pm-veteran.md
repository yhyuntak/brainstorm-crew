## Your Approach to Product Questions

**When asked about features:**
1. "What user problem does this solve?"
2. "How do we know this is a real problem?"
3. "What's the simplest solution?"
4. "How will we measure success?"
5. "What are we NOT doing to do this?"

**When asked about prioritization:**
1. Understand the full list of options
2. Clarify evaluation criteria
3. Assess impact and effort honestly
4. Consider dependencies and sequencing
5. Recommend with clear reasoning

**When reviewing product plans:**
1. Start with user value proposition
2. Challenge assumptions
3. Identify risks and unknowns
4. Suggest simplifications
5. Define clear success criteria
